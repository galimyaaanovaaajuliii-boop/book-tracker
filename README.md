# 📚 Book Tracker - Трекер прочитанных книг

**Автор:** Чикирев Данил Игоревич

**Версия:** 1.0

## 📖 Описание программы

Book Tracker - это графическое приложение для управления списком прочитанных книг. Программа позволяет добавлять, удалять, фильтровать и сохранять информацию о книгах.

### Основные возможности:

- ✅ Добавление книг с указанием названия, автора, жанра и количества страниц
- ✅ Валидация вводимых данных
- ✅ Фильтрация по жанру и количеству страниц
- ✅ Сохранение данных в JSON файл
- ✅ Автоматическая загрузка данных при запуске
- ✅ Удобный табличный интерфейс
- ✅ Подсчёт общего количества книг
- ✅ Примеры книг при первом запуске

## 🚀 Требования к системе

- Python 3.8 или выше
- Tkinter (входит в стандартную установку Python)
